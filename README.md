# recipe-app 
A recipe searching app that utilizes the food2fork API.

# Technologies Used
React, React-Router, Bootstrap, Food2Fork API

## Getting Started

### Installing

1. Clone the repo

```
git clone https://github.com/zarekivey/recipe-app.git
```

2. Run npm install inside project root

```
npm install
```

3. Start the server

```
npm start
```

4. Explore

```
Enjoy :)
```
